今天，**Sideline** 的官方网站正式上线了！

## 关于网站

网站使用 **Jekyll** 和 **GitHub Pages** 构建，采用像素游戏风格设计，主要包含以下模块：

- **🏠 首页** - 游戏介绍、特性展示、下载入口
- **📖 关于游戏** - 核心玩法、设计理念、开发路线图
- **📚 开发文档** - 技术架构、API 参考、构建指南
- **📝 更新日志** - 开发进展、版本更新记录

## 技术栈

| 技术 | 用途 |
|------|------|
| Jekyll | 静态网站生成器 |
| GitHub Pages | 免费托管 |
| 像素风格 CSS | 自定义主题 |
| Google Fonts | ZCOOL KuaiLe 中文像素字体 |
